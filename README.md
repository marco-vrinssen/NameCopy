# Copy All The Names

World of Warcraft Retail addon that allows you to copy player names from context menus throughout the game.

## Features

Right-click on any player name to copy their full Name-Realm format. Works in:

- Player unit frames (party, raid, target, focus)
- PVP scoreboards (battlegrounds, arenas)
- Guild and community members
- Friends list and Battle.net friends
- Chat messages and whispers
- Group Finder (LFG/LFR)

## Usage

1. Right-click on any player name in the game
2. Select "Copy Full Name" from the context menu
3. A dialog appears with the full Name-Realm format
4. Press Ctrl+C (Cmd+C on macOS) to copy
